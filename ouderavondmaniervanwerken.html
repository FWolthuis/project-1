Manier Van Werken

Wij werken voornamelijk op de laptop met behulp van boeken als het nodig is.
Als we op de laptop werken maken we gebruik van licenties om ons werk te kunnen maken.
We maken dan ook Engels, Nederlands, Rekenen en Burgerschap allemaal via de laptop.
Voor het programeren gebruiken we een programma waar wij onze code in kunnen zetten en daarna ook in kunnen testen,
bijv. VS Code of Notepad++.
